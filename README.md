# Wonosobo

![Wonosobo](https://raw.githubusercontent.com/indogegewepe/Wonosobo/refs/heads/main/assets/img/Untitled.png)

## Description

Wonosobo is a web application designed to showcase the beauty and attractions of Wonosobo, a regency in Central Java, Indonesia. This project aims to provide visitors with comprehensive information about tourist destinations, local culture, and travel guides.

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Hosting: Great-Site.net

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/indogegewepe/Wonosobo.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Wonosobo
   ```

3. Set up your database configuration in the appropriate configuration file.

4. Import the database schema if available.

5. Configure your web server to point to the project's public directory.

## Usage

To run the application locally:

1. Start your local web server (e.g., Apache, Nginx).

2. Ensure your database server is running.

3. Access the application through your web browser at `http://localhost`.
